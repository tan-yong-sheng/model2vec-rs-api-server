THE FIX (in modal_deploy.py):

  BEFORE (Broken):
    proc = subprocess.Popen(["/app/model2vec-api"], env=env)
    proc.wait()  # ❌ Blocks forever, starves heartbeat

  AFTER (Fixed):
    proc = subprocess.Popen(["/app/model2vec-api"], env=env)
    try:
        while proc.poll() is None:
            time.sleep(0.5)  # ✅ Yields every 500ms, heartbeat can run
        if proc.returncode != 0:
            raise RuntimeError(...)
    finally:
        # Graceful cleanup

BENEFIT:
  - Heartbeat gets CPU time every 500ms
  - No more "Runner terminated" crashes
  - Container stays alive indefinitely
  - Clean shutdown on process exit
